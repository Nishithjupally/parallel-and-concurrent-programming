Readme file for Comparing Different Parallel Implementations for Identifying Prime Numbers:

'input-param.txt';

$ g++ Src-CS18BTECH11018.cpp -lpthread

$ ./a.out

Output files
1.Primes-DAM.txt - primes produced by dam
2.Primes-SAM.txt - primes produced by sam
3.Times.txt -time taken by dam and sam respectively
